ISLT_4631

Introduction and Learning Objectives
The objective of this project was to create voice over audio recordings of all long form paragraphs throughout my online portfolio. This was done in order to increase the accessibility of the website by creating an audio reading of each text section, so that users can have the audio read to them if needed. I will be implementing these readings by inserting a speaker icon, in the corner of each text box, that users can click on to open an audio player and listen to the text.

Licensing
The only unoriginal audio is a track named "Drifting at 432 Hz" created by Unicorn Heads. This track is a free, commercial use track provided by the public YouTube Audio Library that requires no attribution in online usage.

Techniques
•Changed Track Decibel Volumes
•Inserted Exponential Fade Out
•Inserted Background Ambient Audio
•Utilized Natural Fade In using Music
•Trimmed and Spaced Audio Layers
•Applied Click Eliminator Process

Reflection
Creating audio forms for each text exert from the website was helpful not only to provide assistance to users, but also to demonstrate other forms of embedding that can be completed. This project allows me to improve my website as well as demonstrate another skill in web development to potential clients.
